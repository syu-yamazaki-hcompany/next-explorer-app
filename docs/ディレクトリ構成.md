# ディレクトリ構成
コンポーネントはもっと整理したほうがいいと思いましたが、
少ないのでそのままにしちゃいました。
親と子ぐらいでまとめたほうがよかったかもしれないです。

主なディレクトリ構成
```sh
app/
├── user/
│   └── [login]/page.tsx      # 詳細情報API、詳細情報表示（サーバー）
├── page.tsx                  # 検索REST API（サーバー）
├── layout.tsx                # 共通のレイアウト（サーバー）
├── loading.tsx               # 共通のローディング画面（サーバー）   
│
components/
│   ├──BackButton.tsx         # 詳細ページから検索画面に戻るボタン(クライアント)
│   ├──RepoCard.tsx           # 詳細ページに表示するリポジトリカード（サーバー）
│   ├──ScrollReset.tsx        # 検索画面から遷移した際のスクロールをリセット（クライアント）
│   ├──ScrollToTopButton.tsx. # スクロールトップボタン（クライアント）
│   ├──SearchForm.tsx         # 検索フォーム（クライアント）
│   ├──SearchPage.tsx         # 検索画面でスケルトンUIなどの状態を持つページ（クライアント）
│   ├──ThemeToggle.tsx.       # テーマ切り替えボタン（クライアント）
│   └──UserCard.tsx           # 検索画面の結果一覧表示カード（クライアント）
│
lib
│ └── graphqlClient.ts        # GraphQLクエリ集約場所
styles/
│ └── globals.css　　　　　　   # 共通のスタイリング設定
```
